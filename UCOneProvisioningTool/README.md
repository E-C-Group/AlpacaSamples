# UCOne Provisioning Tool

## Prerequisite
- Modify the `SERVICEPACKS` variable to include the Service Packs that are required.
    - `private static final String[] SERVICEPACKS = {"ServicePack1", "ServicePack2"};`
    
## Usage 
- `./runner UCOneProvisioningTool.java <UserID>`
- Where `UserID` is the BroadWorks UserID of the User to provision UC One for.
